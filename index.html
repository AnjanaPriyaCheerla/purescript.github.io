PureScript Community Blog

Welcome

PureScript is a small strongly, statically typed programming language with expressive types, written in and inspired by Haskell, and compiling to Javascript.

This community blog is a collection of posts written by the authors and users of the language. If you are interested in submitting an entry, you can submit a pull request on GitHub.
